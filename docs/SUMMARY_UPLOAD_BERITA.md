## ğŸ” TROUBLESHOOTING

### Problem: "Gagal mengupload gambar"
Solution:
- Cek permission folder uploads (harus writable)
- Pastikan folder uploads ada
- Cek php.ini: upload_max_filesize = 10M

### Problem: Tidak ada respon setelah klik Simpan
Solution:
- Cek browser console (F12) untuk error
- Pastikan sudah login sebagai admin
- Cek error log PHP di XAMPP

### Problem: Berita tidak muncul di daftar
Solution:
- Cek tabel articles di database
- Refresh halaman admin
- Cek query di admin.php

### Problem: Alert tidak muncul
Solution:
- Hard refresh browser (Ctrl + Shift + R)
- Clear cache
- Buka di incognito mode
